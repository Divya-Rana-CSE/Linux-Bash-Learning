# Linux Bash Learning

A repository documenting my journey learning Linux and Bash scripting fundamentals.

## 📚 Contents

### DAY1 - Basic Commands
Introduction to essential Linux/Bash commands and basic shell scripting.

**Files:**
- `Basic_Commands.md` - Documentation of basic Linux commands and their usage
- `bs` - Bash script demonstrating basic shell operations
- `myscript.sh` - Custom shell script for practice
- `practice.sh` - Practice exercises for shell scripting

### DAY2 - Variables
Working with variables in bash scripts, command substitution, and environment variables.

**Files:**
- `Variables.md` - Documentation of variable declaration, usage, and quoting rules
- `practice` - Bash script with each variable command used
- `myscript` - Practice script demonstrating variable usage
- `uses` - Practice script showing real-world variable applications

## 🚀 Getting Started

To use these scripts, you'll need:
- A Linux/Unix environment or Git Bash on Windows
- Bash shell
- Basic understanding of command line

## 📖 How to Use

1. Navigate to the DAY1 folder:
   ```bash
   cd DAY1
   ```

2. Review `Basic_Commands.md` for command documentation

3. Run the practice scripts:
   ```bash
   bash myscript.sh
   bash practice.sh
   ```

4. Explore and modify scripts to learn!

## 📝 Topics Covered

- Basic Linux commands and file operations
- Shell script basics
- Working with bash variables and functions
- Command execution and scripting fundamentals
- Variable declaration and usage
- Quoting rules (single vs double quotes)
- Command substitution with $()
